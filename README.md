# Artemis Builder (terminal)

A small terminal-based building game where you hunt for spacecraft components and assemble the Artemis spacecraft. When assembled correctly the craft launches and you run experiments in lunar orbit.

How to run:

1. Ensure you have Python 3.8+ installed.
2. In the project folder run:

```bash
python index.py
```

Gameplay:
- Move with `w`, `a`, `s`, `d`.
- Components are placed on the map (single-letter shown). Move onto a component to pick it up.
- Return to the launchpad (center tile `L`) and use `place <component>` to assemble.
- When all components are placed the spacecraft launches and you can run experiments in orbit.

Files:
- `index.py`: game source.

Enjoy building Artemis!
